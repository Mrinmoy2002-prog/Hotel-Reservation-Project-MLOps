Hotel Reservation Prediction

Welcome to the Hotel Reservation Prediction project! This project aims to predict whether a hotel reservation will be canceled or not based on various features of the reservation and the customer.

Project Overview

The project involves data preprocessing, exploratory data analysis, feature engineering, model training, and evaluation. We use a dataset containing information about hotel reservations, including details such as booking date, arrival date, number of guests, and more.

Technologies Used

  • Python
  • Pandas
  • Scikit-learn
  • LightGBM
  • MLflow

Getting Started

To get started with this project, you can clone the repository and follow the instructions in the README file. The project is structured into different modules for data preprocessing, model training, and evaluation.
